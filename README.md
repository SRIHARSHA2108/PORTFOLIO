# SRIHARSHA Portfolio

A professional single-page portfolio website for SRIHARSHA, built to present software and hardware skills, major/minor projects, and contact information.

## Overview

This portfolio highlights work across both software development and embedded hardware. It includes a modern hero section, project showcase, skills section, contact section, section-wise background images, reveal animations, and a project detail popup for the Farmer Market Platform.

## Projects

- Farmer Market Platform - Major Project
- Smart Irrigation Wireless Remote Control - Major Project
- Space Crew Mission - Minor Project

## Technologies Used

- HTML5
- Advanced CSS
- Bootstrap 5.3.8
- Tailwind CSS browser CDN
- JavaScript
- Python and Flask mentioned as backend skills
- Embedded C++, Raspberry Pi, ESP32, DHT sensors, and IoT sensors mentioned as hardware skills

## Features

- Responsive single-page layout
- Glass-style fixed navigation bar
- Section-wise background images
- Animated project cards
- Scroll reveal animation using JavaScript
- Bootstrap modal popup for project details
- Vercel demo button for Farmer Market Platform
- Skills section for software and hardware technologies

## How To Run

Open `index.html` directly in a web browser.

No build command or local server is required because the page uses CDN links for Bootstrap and Tailwind CSS.
